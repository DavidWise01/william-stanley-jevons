# WILLIAM STANLEY JEVONS

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **LOGIKĒ**.

WILLIAM STANLEY JEVONS (1835–1882) — LOGIKĒ, the BOOLEAN CLEANUP (alongside [[george-boole]]/[[augustus-de-morgan]]). Boole's algebra was quirky (uninterpretable terms, exclusive 'or', meaningless division); Jevons CLEANED IT UP — made 'or' INCLUSIVE, enshrined the idempotent law x+x=x, and recast inference as ELIMINATION over a 'logical alphabet' of every term-combination. Then he built ⚑ THE LOGIC PIANO (1866) — a wooden keyboard machine that takes premises and, by knocking out every combination they forbid, DISPLAYS the valid conclusions on its face: the FIRST machine ever to solve a problem of formal logic, a Boolean computer 70 years before electronics (it survives in Oxford's museum). LIVE: play the Logic Piano — the 8 term-combinations (A,B,C) as keys; toggle premises and watch contradictory worlds go dark; the survivors ARE the conclusion (verified: All A→B + All B→C derives All A→C, Barbara, mechanically). CONFIDENCE-TIERED: STRONG (a real working logic machine + a sound cleanup of Boole — inclusive OR, idempotency, elimination) / A-DEMONSTRATOR (handles a few terms — proves logic CAN be machined, not a general computer; the Boolean-algebra=switching-circuits leap is [[claude-shannon]]'s, 70 yrs on) / LOGIC-WAS-HIS-SIDELINE (better known as an ECONOMIST — co-founder of the marginal-utility revolution, namesake of the 'Jevons paradox'; drowned swimming at 46). ⚑ The first time inference left the human head and entered a device — the gate, in its first wooden draft.
